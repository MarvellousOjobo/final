FORM ATTRIBUTES

We have learnt about attributes and table attributes but forms also have special attributes that help it function better. There are three(3) atributes that we have considered important which are ;
  • required.
  • readonly.
  • placeholder.

Required attribute

It is used tell HTML that data must be entered in the input field that the attribute is attached to.

EXAMPLE
< form>
< input type="text" required>
< input type="submit" value="SUBMIT">
OUTPUT
Now try clicking on the SUBMIT button without entering any data in the field and see what happens.

Readonly attribute

It creates an input field which does not accept data input. It may seem useless but in cases were you're creating things like a calculator were data is not entered but the numbers are being clicked and they appear in the input field.

EXAMPLE
< input type="password" readonly>

Now data cannot be entered in the input field.

Placeholder attribute

It is used to tell the user what to input in the input field.

EXAMPLE
< input type="password" placeholder="Please enter your password">

Now in the example, there is a text in the input box telling the user to input his/her password.
