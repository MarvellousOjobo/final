HTML ATTRIBUTES

A HTML attribute just like a normal attribute tells more information about a HTML element and still modifies it.

Now this is an example which aligns the text in the p tag to the right.

The attribute is the example above is (align="right") where "align" is the name and "right" is the value. The equal to sign "=" asigns the value to the name so it is very important.
MARVEL: Don't forget, the attribute is always written in the opening tag not the closing tag.
