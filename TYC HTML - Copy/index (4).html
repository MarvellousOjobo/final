MEASUREMENT ATTRIBUTES

Attributes are also used for sizing HTML elements.

There are two main quantities used for sizing HTML elements;
  • Height-It describes how high or long the element would be.
  • Width-It descibed how wide the element would be.

There are also two main units for sizing HTML elements.

  • pixels(px)
  • percentage(%)
