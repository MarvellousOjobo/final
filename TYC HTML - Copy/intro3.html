WORKING WITH TEXT EDITORS

Text editors are programs which help us convert our HTML code into useful output that appears as our website. Over the years many text editors have been developed such as NOTEPAD, SUBLIME TEXT EDITOR, DCODER, QUICKEDIT, e.t.c.

To run a HTML file you have to save the file with an extension which is .html or .htm

When your code is run,you can see the output as it would show on the web.

MARVEL:Try different text editors and see which one works best for you.
