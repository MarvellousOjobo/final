So in the code in the previous examples, there were tags used to create the form. We are going to analyze these tags. There are;
  • FORM TAG.
  • LABEL TAG.
  • INPUT TAG.
FORM TAG
The form tag is used to span a form. It has an opening and closing tag.

LABEL TAG
The label tag as the name implies creates a label for the input field. Examples of labels are username, password, e.t.c.It has a closing tag.

INPUT TAG
The input tag is what does all of the magic of forms. It spans the input field so that data can be entered. It does not have a closing tag. In the previous example you would see a code like this, input type="text". This is an example of an input syntax. If you have worked with Microsoft Access 2007, you should be familiar with data types. Examples of data types are text,number,e.t.c. In the example above, the data type used is"text". This means that texts are allowed to be entered. HTML has many other data types;
  • text
  • number
  • date
  • color
  • date
  • datetime
  • email
  • month
  • range
  • search
  • tel
  • time
  • url
  • week
  • submit

MARVEL: Another example is using the "number" data type(input type="number"). This allows only numbers to be entered in the input field. The submit attribute (input type="submit") creates a button which submit the inputted data provided it has a direction to be submitted to.
