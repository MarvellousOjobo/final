HTML TABLES

A table as we know is a collection of rows ansd columns in which data is stored in. HTML can also be used to create these tables.

To understand HTML tables there are three(3) important tags to learn first;

  • table tag- This is what spans the whole table.
  • tr tag- The table is divided in table rows(tr).
  • td tag- Table rows are also divided into table data(td)
