ADDING VIDEOS TO WEBPAGES.

  1. STEP 1: Create a folder e.g TYC
  2. STEP 2: Add any video of your choice in the folder.
  3. STEP 3: Rename the video without erasing the video extension(mp4) e.g marvel.mp4
  4. STEP 4: Create a HTML file and save it in the folder you created earlier.
  5. STEP 5: In your HTML file input this code < video src="video name and extension" controls> and the closing tag.

    e.g < video src="marvel.mp4" controls> and the closing tag.

    OR
    < video controls>
    < source src="marvel.mp4" type="video/mp4">
    < source src="marvel.ogg" type="video/ogg"> Video is not supported by your browser
    AND THEN YOU ADD THE CLOSING TAG.

    In this case, when the first file format(mp4) isn't supported the second file format(ogg) gets used.

MARVEL:The three currently supported file extensions for the audio elements are MP3,WAV and OGG.
