ADDING AUDIO TO A WEBPAGE.

  1. STEP 1: Create a folder e.g TYC
  2. STEP 2: Add any audio of your choice in the folder.
  3. STEP 3: Rename the audio without erasing the audio extension(mp3) e.g marvel.mp3
  4. STEP 4: Create a HTML file and save it in the folder you created earlier.
  5. STEP 5: In your HTML file input this code < audio src="audio name and extension" controls> and the closing tag.

    e.g < audio src="marvel.mp3" controls> and the closing tag.
