THE HTML TAG(The Bun)

It is made up opening and closing tags. It begins every HTML code and also ends or concludes it. The HTML tag is a very important tag because it starts and ends the HTML code.

SYNTAX

THE HEAD TAG(The fillings)

This is the part which contains the title and heading of the website. It contains most of the non-visual content that help the website function. It is also useful when linking the HTML file to the CSS file or JavaScript. You'll learn about that in the CSS and JavaScript courses. It is placed in the HTML tag.

SYNTAX

THE BODY TAG(The Ham)

Now this is where all the visual content of your website is being laid. Anything you want to be shown to be shown in your website is written here except things like the title. It is also placed in the body tag after the head tag.
