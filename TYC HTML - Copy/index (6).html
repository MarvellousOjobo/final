AUDIO FEATURES ON HTML

There are certain features that make audio play on HTML more awesome. These are;

  • controls-this activates controls like play,pause,e.t.c.(< audio controls>)
  • autoplay-this allows the video play when ready without the user clicking on play.(< audio controls autoplay>)
  • autoplay loop- this makes the video replay when it ends.(< audio controls autoplay loop>)

MARVEL:The three currently supported file extensions for the audio elements are MP3,WAV and OGG.
